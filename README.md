# VMG Active — Immersive UX Case Study

A static, GitHub Pages-ready portfolio case study.

## Included
- Immersive loading experience
- Scroll-responsive floating product visual
- Fixed chapter navigation
- Mouse parallax and magnetic CTAs
- Interactive design-decision accordion
- Tilt interactions for cards and system components
- Animated journey progress and metrics
- Responsive mobile treatment
- Real VMG Active visuals stored locally
- Working external links

## Publish on GitHub Pages
Upload all files and the complete `assets` folder to the repository root. In **Settings → Pages**, choose **Deploy from a branch**, then select `main` and `/(root)`.

## Contact-link check
Search `mailto:` in `index.html` to replace the email if needed.
